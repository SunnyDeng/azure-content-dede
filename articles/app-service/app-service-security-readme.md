# Azure App Service-Sicherheit

Azure App Services erfüllt die wichtigsten Industriestandards für Sicherheit und Zuverlässigkeit, um unseren Kunden eine sichere Plattform bereitzustellen. Wir bieten Ihnen für Ihre App zahlreiche benutzerfreundliche Funktionen zum Schutz von eingehendem und ausgehendem Datenverkehr. Wir unterstützen unsere auch Kunden dabei, ihren Anwendungscode zu schützen, indem wir extern bereitgestellte Funktionen zur Verfügung stellen, die Ihre Webanwendungen nach Schwachstellen durchsuchen.
