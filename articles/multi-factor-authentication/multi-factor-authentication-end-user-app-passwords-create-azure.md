# Erstellen von App-Kennwörtern für Azure Multi-Factor Authentication im Azure-Portal

Wenn Sie Multi-Factor Authentication mit Azure verwenden, können Sie App-Kennwörter über das Azure-Portal erstellen.

## So erstellen Sie App-Kennwörter im Azure-Portal

1. Melden Sie sich beim Azure-Verwaltungsportal an.
3. Klicken Sie oben mit der rechten Maustaste auf Ihren Benutzernamen, und wählen Sie "Zusätzliche Sicherheitsüberprüfung" aus.
5. Wählen Sie oben auf der Seite "Proofup" die Option "App-Kennwörter" aus.
6. Klicken Sie auf **Erstellen**.
7. Geben Sie einen Namen für das App-Kennwort an, und klicken Sie auf **Weiter**.
8. Kopieren Sie das App-Kennwort in die Zwischenablage, und fügen Sie es in Ihrer App ein.
